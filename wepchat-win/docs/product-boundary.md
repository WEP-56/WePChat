# WePChat Windows — 产品边界与定位

> 状态：**v0.2 已拍板**（开工约束文档）  
> 工作区：`e:\wepchat\wepchat\wepchat-win`  
> 演示图：`wepchat-win/image/`（若缺失请从备份补回）  
> 更新：2026-07-21

---

## 1. 一句话定位

**桌面版的「舒适聊天盒子」**：把更好、更可用的 ChatGPT 式体验放在 Windows 本地。  
轻量、克制、快捷。用来快速测 Key、快速问答、快速验证模型可用性——**不是**另一个 Codex / Claude Code / 本地 IDE Agent。

**品牌：WePChat**（与安卓同一产品名）。

---

## 2. 核心目的

1. 快速测试 Key 联通性  
2. 快速问答  
3. 快速询问问题  
4. 快速验证可用性  

任何功能先问：是让聊天更快更舒服，还是在做编程 Agent？后者默认不做。

---

## 3. 已拍板决策

| # | 议题 | 结论 |
|---|------|------|
| 1 | 品牌 | **WePChat** |
| 2 | 备份 | **与安卓互通**（`.wepchat` v1） |
| 3 | Codex 远控 | **Windows 不做** |
| 4 | 文件根 | **有默认值 + 设置可改**；新会话工作区建在该目录下 |
| 5 | 工具 | **简易工具**；公开名/参数与安卓**契约兼容**，执行层 Windows 自研（Rust FS 等） |
| 6 | HTML 等 | **尽量完善** + 流式演示 |
| 7 | 供应商 | 完整多供应商；**OpenAI Chat / Responses / Anthropic**；**HTTP 出站走 Rust** |
| 8 | 前端 | **专用前端**；协议适配可参考安卓逻辑，**不跑安卓运行时**（无 plus/uni / session.files） |
| 9 | 分发 | **安装包 + 便携**；GitHub Actions |
| 10 | 自动更新 | 默认 **M4** |

---

## 4. 我们是 / 不是

**是**：本地多供应商聊天、会话管理、Markdown、会话文件区、简易生图、HTML/JS 轻验证、`.wepchat` 互通。

**不是**：Codex/Claude Code、Win 端远控、真 Shell、完整 IDE、Cherry 全家桶。

---

## 5. 界面骨架（M0 已落地）

- **最左图标轨**：聊天 · 生图 · 设置（底）
- **偏左列表栏**：会话列表 / 生图任务 / 设置分类
- **中部**：对话区 / 生图区 / 设置页
- **右侧多侧栏**：流式预览 · 工作区文件 · 文件查看（可关）
- 原则：高级简约、配色纯净、初始内容尽量少

---

## 6. 文件根

- 默认：`文档/WePChat/workspaces`
- 设置可改自定义目录
- 新会话：`{workspaceRoot}/{sessionId}/`
- 工具路径限制在当前会话工作区内

---

## 7. 共识摘要

```text
WePChat Windows = 本地舒适聊天盒子（Tauri + 专用前端）
目的 = 测 Key · 快问 · 快答 · 验可用性
不要 = Codex 远控 · IDE Agent · 真 Shell
```
